Sources &
Methodology

EthicPrint scores brands using verified, publicly available sources. Not all sources carry the same weight — here's exactly how we decide which publishers we trust and how much.

The tier system
T1
Primary sources
International NGOs, investigative media, and recognized research institutes.
weight ×3
T2
Verified sources
National media, official government data, institutional and certified reports.
weight ×2
T3
Supporting sources
Blogs and secondary publications. Valid only as supporting evidence.
weight ×1
Confidence levels — per category
High
Weighted score ≥ 6  ·  e.g. 2× Tier 1, or 1× T1 + 3× T3
Medium
Weighted score ≥ 3  ·  a single Tier 1 source already qualifies
Low
Weighted score ≥ 1  ·  limited evidence, treat with caution
None
No verified sources yet for this category

Tier 3 · Supporting weight ×1 ▼

Tier 3 includes blogs, specialist publications, and secondary sources not covered by Tier 1 or Tier 2. These are used only as supporting evidence alongside higher-tier sources — never as the sole basis for a score. They are evaluated case-by-case and do not have a fixed whitelist.

Every score change requires human review. New sources — whether found automatically or proposed by the community — are never applied directly. They go through a manual approval step before affecting any brand score. If you know a reliable source we're missing, write to us.
